   - [Bad Auction](contracts/BadAuction.sol) contract
     -  [Attacking Bad Auction](contracts/AttackBadAuction.sol)
#### Quick start
1. starting TestNet

In new terminal start `ganache-cli`. 

```bash
    $ ganache-cli
```
Using terminal 2 install and migrate the following contracts
```bash
    $ npm install
    $ npm run compile
    $ npm run migrate 
```
