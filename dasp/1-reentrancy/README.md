#### Quick start
##### 1. starting TestNet

Open a new terminal in order to start start `ganache-cli`. 

```bash
    $ ganache-cli
```

##### 2. Using terminal 2 
install, compile and migrate the following contracts
```bash
    $ npm install
    $ npm run compile
    $ npm run migrate 
```
